Guía de Usuario - CRM v1.0.0
Para la guía completa, consulta USER_GUIDE.md en la carpeta del proyecto.
Gestión de Clientes
  • Crear: Botón "Nuevo Cliente" → Completar formulario (solo nombre obligatorio)
  • Editar: Abrir detalle → Botón "Editar" (✏️)
  • Archivar: Botón 🗑️ (reversible, no elimina físicamente)
  • Navegación Kanban: "Ver en Kanban" resalta oportunidades del cliente
Gestión de Contactos
  • Añadir: En detalle cliente → Sección Contactos → "Añadir"
  • Múltiples canales: Click "+ Añadir" para más emails/teléfonos
  • Principal: Click ⭐ para marcar email/teléfono principal
  • Editar: Botón ✏️ → Modificar datos, añadir/quitar canales
  • Eliminar: Botón 🗑️ para eliminar canal o contacto completo
Búsqueda y Filtros
  • Búsqueda: Escribe en barra superior (busca en nombre, email, teléfono, CIF)
  • Filtros combinables: Estado, Tipo, Región, Fuente, Responsable
  • Estado: Activos / Archivados / Todos
Campos del Cliente

Obligatorios:

  • Nombre

Opcionales:

  • Web, Teléfono, Email, CIF/NIF
  • Dirección, Tipo, Región, Fuente
  • Responsable, Notas
Consejos:
  • Website acepta formato simple: www.ejemplo.com (se añade https:// auto)
  • Asigna responsable a cada cliente para mejor seguimiento
  • Marca siempre un email/teléfono como principal
  • Usa notas para contexto importante del cliente
