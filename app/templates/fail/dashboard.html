Guía de Usuario - CRM v1.0.0
Para la guía completa en detalle, consulta el archivo USER_GUIDE.md en la carpeta del proyecto.
Dashboard
  • KPIs: Pipeline Total, Pipeline Ponderado, Cerrado Anual, Objetivos
  • Kanban: Arrastra tarjetas entre columnas para cambiar etapa
  • Filtros: Región, Tipo, Responsable, Búsqueda
Clientes
  • Crear: Botón "Nuevo Cliente" → Completar formulario
  • Editar: Abrir detalle → Botón "Editar"
  • Contactos: Múltiples por cliente con emails/teléfonos
  • Navegación: "Ver en Kanban" resalta oportunidades del cliente
Contactos
  • Añadir: En detalle de cliente → "Añadir" contacto
  • Múltiples canales: Emails y teléfonos ilimitados
  • Principal: Marca con ⭐ el email/teléfono principal
  • Eliminar: Botón 🗑️ para quitar canales o contacto completo
Kanban
  • Etapas: New → Contacted → Qualified → Proposal → Negotiation
  • Mover: Arrastra y suelta para cambiar etapa
  • Cerrar: Arrastra a Won (ganada) o Lost (perdida)
  • Ocultar: Activa "Ocultar cerradas" para enfocarte en activas
Importar Excel
  • Plantilla: Usa IMPORT_NORMALIZADO_CRM.xlsx
  • Sheet: Debe llamarse "DATA"
  • Validación: Sistema valida antes de importar
  • Reporte: Muestra warnings y errores
Consejos Rápidos
  • Usa filtros para encontrar rápido
  • Asigna responsable a cada cliente
  • Mantén contactos actualizados
  • Revisa Kanban diariamente
  • Pipeline saludable: 20-25% en cada etapa
  • Marca email/teléfono principal
  • Usa notas para contexto importante
  • Archivar != Eliminar (es reversible)
Importante:
  • Los datos se guardan automáticamente al cerrar modals
  • Archivar no elimina físicamente (se puede recuperar)
  • Cambiar contraseña en tu primer uso

v1.0.0 - Release Inicial (16 Enero 2026)

Primera versión completa y estable para producción interna.

✨ Características Principales
  • Gestión de Clientes: CRUD completo con 11 campos configurables
  • Gestión de Contactos: Múltiples emails/teléfonos, roles, marcado de principal
  • Dashboard & KPIs: Pipeline Total/Ponderado, Cerrado Anual, Objetivos
  • Kanban Visual: Drag & drop, 5 etapas, filtros múltiples
  • Importación Excel: Validación automática, reporte detallado
  • Seguridad: 3 roles (admin/sales/viewer), JWT, auditoría
🔧 Stack Técnico
  • Backend: FastAPI + SQLAlchemy + SQLite
  • Frontend: HTML5 + Bootstrap 5 + JavaScript vanilla
  • Auth: JWT con HttpOnly cookies
  • BD: SQLite (compatible PostgreSQL)
⚠️ Limitaciones Conocidas
  • Export de datos no implementado (usar herramientas externas)
  • Solo EUR soportado
  • Notificaciones: Planeado para v1.2.0
